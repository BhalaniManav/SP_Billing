Share on Local WiFi (One-Click Method)

For security, a browser can't start a server automatically. Instead, create a simple helper file for one-click access:

  1. In the same folder as this HTML file, create a new file named start_server.bat
  2. Open it in Notepad and copy the code below into the file. Save it.
  3. Double-click `start_server.bat`. It will show your IP and start the server.
@echo off
echo.
echo ===========================================
echo      Shital Paan Local Network Server
echo ===========================================
echo.
echo Your Computer's IP Addresses are:
ipconfig | findstr /i "IPv4"
echo.
echo ===========================================
echo. 
echo ==> Server starting now...
echo ==> Open a browser on your phone and go to:
echo      http://[YOUR-IPv4-ADDRESS]:8000
echo.
python -m http.server 8000
